# NUMERIS – NGX Investment Intelligence Platform

NUMERIS is a financial intelligence SaaS platform designed to democratize access to Nigerian Stock Exchange (NGX) data.  
It transforms raw NGX filings into clean, structured, and comparable insights for retail and institutional investors.

---

## 🚀 MVP Scope (Jan 15 – Feb 15)
The MVP demonstrates that NGX financial data can be:
- Extracted, parsed, and normalized into a structured database
- Converted into ratios and trend analysis
- Evaluated through a rules-based intelligence layer
- Delivered via a self-service web dashboard

---

## 🎯 Core Features
- 5-year financial performance for listed NGX companies
- Profitability, leverage, and capital efficiency metrics
- Buy / Hold / Avoid signals based on fundamentals
- Sector benchmarking and red/green flag detection

---

## 🛠 Tech Stack
- **Python** (data extraction, parsing, automation)
- **PostgreSQL** (financial database)
- **Pandas / Numpy** (data cleaning & normalization)
- **Camelot / Tabula / pdfplumber** (PDF parsing)
- **SQLAlchemy** (database integration)
